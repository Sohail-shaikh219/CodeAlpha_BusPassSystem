# 🚌 Cloud-Based Bus Pass System

This is a cloud-based web application that allows users to generate digital bus passes instantly.  
Developed as part of my Cloud Computing Internship.

---

## 🚀 Features
- User-friendly form to enter details
- Instant bus pass generation
- Clean and responsive UI
- Cloud deployment for global access

---

## 🌐 Live Demo
👉 https://codealpha-buspasssystem.onrender.com

---

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS
- Render (Cloud Hosting)

---

## 📌 Project Description
This system allows users to enter their details such as name, route, and date, and generates a digital bus pass dynamically.  
The application is deployed on the cloud, ensuring accessibility and scalability.
